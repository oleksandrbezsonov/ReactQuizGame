# ReactQuizGame
Totally a quiz game

- Progress Bar;
- Game logic and ability to restart;
- Staged rendering;
- UseState hook usage;

#Install
NodeJS must be installed.
 Clone the repository via bash and run the following commands:
 ```shell
npm install
npm start
```
